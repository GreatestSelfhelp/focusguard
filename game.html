🎮 1. โค้ดเกม

เกมต้องรันได้ในตัวเอง (ไม่เรียก API ของ Gemini สด) — เกมแบบตรรกะ/คำถาม/อาร์เคดทำงานได้เต็มที่ เกมที่คุยกับ AI สดๆ จะใช้ไม่ได้เพราะไม่มีกุญแจ API ติดไปด้วย

⚙️ 2. ตั้งกติกา

"ผ่อนผัน" = สลับแท็บสั้นๆ (เช่นแตะแจ้งเตือน) ไม่นับผิด ถ้านานกว่านี้ถึงจะโดนหัก

✅ ลิงก์พร้อมแล้ว

ลิงก์นี้ยาว (เกมทั้งเกมอยู่ในลิงก์) — ไม่มี QR ให้สแกน ส่งผ่าน Google Classroom หรือคัดลอกวางใน Line ได้
รหัสปลดล็อก (เก็บไว้กับครูเท่านั้น)
----
ถ้านักเรียนถูกล็อกหน้าจอ ให้ครูพิมพ์รหัสนี้ให้นักเรียนปลดล็อกเอง
🧪 ทดสอบลิงก์ก่อนแจกจริงเสมอ

🔍 3. ตรวจรหัสยืนยันจากนักเรียน

เกมรันในกรอบแซนด์บ็อกซ์แยกจากหน้าครอบ — โค้ดเกมแตะคะแนน/ระบบล็อกไม่ได้
ป้องกันการใช้อุปกรณ์เครื่องที่สองไม่ได้ — เป็นเครื่องมือช่วยสังเกต ไม่ใช่ระบบรักษาความปลอดภัยเต็มรูปแบบ
